Proiecte

Image Analyser

O aplicație web dezvoltată pentru a ajuta convertirea unui design din imagine în cod.

  • Aplicație centrată pe Canvas, dezvoltată cu ajutorul Canvas API, pentru o mai bună performanță.
  • Teme de noapte și zi, ce îmbunătățesc experiența utilizatorului, folosind CSS3 și proprietăți personalizate CSS.
  • Paletă de culori integrată, cu posibilitatea de a salva culorile în localStorage, folosind Web Storage API.
  • Posibilități de mutare, mărire, măsurare și selectare culori, implementate pentru a ajuta utilizatorul final să își desfășoare treaba într-un mod cât mai plăcut.
  • Scurtături pe tastatură, pentru a crește productivitatea utilizatorului.
  • Suport complet pentru touch.

Health Care

O aplicație web dezvoltată pentru a ajuta convertirea unui design din imagine în cod.

  • Aplicație minimalistă, ce înregistrează date despre sănătatea utilizatorului, bazată pe IndexedDB API.
  • Interfață simplă și intuitivă, proiectată pentru a face experiența utilizatorului mai plăcută.
  • Formulare separate pentru presiunea arterială și ritmul cardiac, respectiv pentru greutate și temperatură, implementate pentru a oferi utilizatorului control complet asupra datelor introduse.
  • Schimbarea zilelor se face printr-o acțiune de glisare de la dreapta la stânga sau invers.
  • Aplicație web progresivă

Recipe Book

O aplicație web dezvoltată pentru a ajuta convertirea unui design din imagine în cod.

  • Proiect realizat în ansamblu - ReactJs, HTML5 și CSS pentru partea de Front-End, NodeJS cu framework-ul Express drept server și JSON ca bază de date.
  • Bară de căutare, sistem de sortare și filtru de categorii, implementate pentru a face experiența utilizatorului cât mai plăcută.
  • Funcție de căutare live, integrată pentru a mări viteza de căutare.
  • Posibilitatea de editare individuală a ingredientelor, instrucțiunilor și notițelor.
  • API simplu, implementat pentru a prelua, crea și șterge rețete din baza de date JSON.
  • Hook personalizat, creat pentru a transmite datele folosind Fetch API.
